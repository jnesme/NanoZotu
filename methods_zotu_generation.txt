Sequence processing and ZOTU generation
========================================

Raw Oxford Nanopore reads (R10.4 flow cell, super-accuracy basecalling) were demultiplexed by
the sequencing instrument and processed per sample. Primer trimming was performed with cutadapt
v5.1 using a linked-adapter strategy (-g 27F...RC_1492R) with the 27F forward primer
(5'-AGRGTTYGATYMTGGCTCAG-3') and the reverse complement of the 1492R reverse primer
(5'-AAGTCGTAACAAGGTARCY-3'). Using linked adapters with --discard-untrimmed requires both
primers to be present on each read, preventing reads with only one adapter from passing. The
--revcomp flag was used to orient all reads in the forward direction in a single pass. Tolerant
alignment settings were applied to accommodate Oxford Nanopore read characteristics and IUPAC
degenerate bases (--error-rate 0.15, --overlap 15). Reads outside the expected amplicon length
range of 1,300-1,800 bp were discarded.

Trimmed reads from all 21 samples were pooled prior to dereplication. Read headers were
annotated with a sample=barcodeXX tag before pooling to preserve sample identity for downstream
abundance table construction. Dereplication was performed with usearch v12
(-fastx_uniques -sizeout), retaining abundance information for each unique sequence.

Denoising was performed with the UNOISE3 algorithm (usearch v12, -unoise3) across a minsize
sweep from 1 to 8 to identify a stable ZOTU threshold. The working threshold was set at
minsize=5, selected based on the inflection point in the ZOTU count curve (463, 215, 141, 118,
101, 89, 76 ZOTUs at minsize 1-7), where the rate of reduction stabilised. A per-sample ZOTU
abundance table was generated by mapping the pooled reads back to the ZOTU sequences using
usearch -otutab, which uses the sample= header tags to assign reads to their originating sample.


Reference database construction
================================

A combined reference database was constructed for taxonomic assignment. Bacterial and archaeal
SSU rRNA representative sequences were downloaded from the Genome Taxonomy Database (GTDB;
bac120_ssu_reps.fna, 88,481 sequences; ar53_ssu_reps.fna, 5,289 sequences). GTDB rank prefixes
(d__, p__, c__, o__, f__, g__, s__) were stripped from taxonomy strings and bracketed locus
metadata discarded, producing semicolon-delimited seven-rank taxonomy strings compatible with
downstream tools. Because GTDB contains exclusively prokaryotic genomic sequences, organellar
16S rRNA sequences — which are amplified by the universal 27F/1492R primers from algal cultures
— are absent from GTDB and would otherwise return the nearest prokaryotic hit (confirmed:
Isochrysis galbana plastid sequences were misassigned to Elusimicrobiota at 88% identity in
initial tests). Chloroplast and mitochondrial sequences were therefore extracted from SILVA 138.2
(4,551 and 1,289 sequences, respectively) and appended to the GTDB SSU sequences, yielding a
combined database of 99,610 sequences. A BLAST nucleotide database was constructed from this
combined FASTA using makeblastdb (BLAST+ v2.16.0, -dbtype nucl).


Taxonomic assignment
====================

ZOTUs were assigned taxonomy by BLASTn search against the combined database (BLAST+ v2.16.0;
-perc_identity 80, -max_target_seqs 500). To ensure identification of the true best hit — since
-max_target_seqs 1 does not guarantee the highest-scoring alignment due to internal database
processing order (Shah et al., 2019, Bioinformatics) — 500 candidate hits were retrieved per
query and the best hit selected by bitscore. ZOTUs with a best-hit percent identity >= 97% and
e-value <= 1e-10 were assigned the corresponding GTDB or SILVA taxonomy. ZOTUs matching
organellar sequences (Chloroplast or Mitochondria lineage) were assigned regardless of percent
identity, as any organellar hit is biologically informative in an algae culture context.
